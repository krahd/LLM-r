# Capabilities

This catalog is generated from `llmr/ableton_osc.py` tool specs.

## Domains

- `song`: transport, tempo, signature, quantization, count-in
- `tracks`: track create/lifecycle/mixer/send
- `session`: scene and clip operations
- `devices`: parameter inspection tools
- `parameters`: parameter writes

Use `GET /api/capabilities` to fetch the runtime source of truth.
